# SUSS OpenClaw — Lead Gen & Profiling Tool

## Status: Brainstorming (In Progress)
**Last updated:** 2026-03-13

## What We're Building
A lead generation and profiling tool for a SUSS school project, to be deployed on OpenClaw.

## Core Features
1. **Find leads** — HR Directors, CFOs in:
   - Oil & gas companies
   - MNCs
   - Tech companies
   - Companies that invest in insurance / health reimbursement packages
2. **Profile each lead** — Pull data from:
   - LinkedIn (primary source)
   - Personal social media (background check)
   - Google search (background check)
3. **Analyze communication style** — Study each lead's language, tone, word choices from their social media posts
4. **Generate personalized outreach** — Craft messages that mirror each lead's communication style to maximize engagement, reply rates, and first meeting bookings
5. **Output pipeline:**
   - Phase 1: Excel spreadsheet (leads + profiles + personalized messaging notes)
   - Phase 2: Dashboard
   - Phase 3: LinkedIn automated outreach integration
